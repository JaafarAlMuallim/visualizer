## Description

This is a simple visualizer for the Data Structures & Sorting Algorithms

- Stack
- Queue
- Linked List
- Insertion Sort
- Bubble Sort
- Quick Sort
- Merge Sort

## Installation

1. Clone the repository
2. run `npm install`
3. run `npm run dev` to run in your development environment
